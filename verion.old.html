1DailyCalendar User Guide

Welcome to 1DailyCalendar

1DailyCalendar is a simple, accessible web application for managing daily notes on a calendar. This guide explains how to use its features effectively.

Features

  • Calendar View: Displays a monthly grid where you can add notes to any day.
  • Note Editor: Add or edit notes with a title and details for each day.
  • Dark/Light Theme: Switch between light and dark modes for comfortable viewing.
  • Search: Find notes by date, title, or content.
  • Import/Export: Save your notes as a JSON file or import them from a previous backup.
  • Today Navigation: Jump to the current month and highlight today's date.
  • Clear All Notes: Remove all notes with a confirmation prompt.
  • Note Counter: Displays the number of notes in the current month.
  • Keyboard Shortcuts: Use shortcuts for quick actions like jumping to today or exporting notes.
  • Accessibility: Keyboard navigation and ARIA support for screen readers.

How to Use

Navigating the Calendar

  • Click the ‚óÄ or ‚ñ∂ buttons to move to the previous or next month.
  • Click the Today button to jump to the current month. Today's date is highlighted with a red border.
  • Click any day to open the note editor, or press Enter or Space when focused on a day.

Adding/Editing Notes

  • In the note editor, enter a title (up to 50 characters) and details.
  • Click Save to store the note, or Close to discard changes.
  • If both title and details are empty, saving will delete the note for that day.

Searching Notes

  • Type in the search bar to find notes by date, title, or content.
  • Click the ‚úñ button to clear the search.
  • Click a search result to jump to that date and open its note editor.
  • Click outside the search bar or results to hide the results.

Importing/Exporting Notes

  • Click Export to download your notes as a JSON file, or press Ctrl + E.
  • Click Import, select a JSON file, and confirm to merge notes into the calendar, or press Ctrl + I.

Clearing All Notes

  • Click Clear All to remove all notes, or press Ctrl + Shift + C.
  • A confirmation prompt will appear to prevent accidental deletion.

Keyboard Shortcuts

  • Ctrl + T: Jump to today.
  • Ctrl + E: Export notes.
  • Ctrl + I: Import notes.
  • Ctrl + Shift + C: Clear all notes.

Troubleshooting

  • Notes not saving? Ensure your browser allows local storage. Check if you're in private/incognito mode, which may disable it.
  • Import failed? Verify the JSON file is valid and matches the expected format (ask for developer support if needed).
  • Calendar looks off? Ensure your browser is up to date and supports modern CSS Grid.
  • Accessibility issues? Contact support to report problems with keyboard navigation or screen readers.
